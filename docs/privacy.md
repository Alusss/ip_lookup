# 隐私合规声明

## 数据收集

- **IP 地址**：仅用于即时展示查询结果，不持久化存储
- **GeoIP 地理位置**（可选功能，默认关闭）：启用时根据 IP 查询城市、国家、ISP 信息，结果即时返回 JSON API 请求，**不记录、不存储、不分享**
- **日志**：记录的 IP 地址已脱敏（IPv4 保留前 3 段，IPv6 保留前 4 组）
- **无 Cookie**：不使用 Cookie、localStorage、或任何客户端跟踪技术
- **无 PII**：不收集任何个人可识别信息（PII）
- **广告关闭**：用户关闭广告后，偏好仅保存在当前会话的 sessionStorage 中，不跨会话跟踪

## 统计埋点

- 优先使用 Cloudflare Web Analytics（无 Cookie、无 PII）
- 不使用 Google Analytics（GA4）或其他需要 Cookie 弹窗的统计工具
- 统计脚本 `defer` 加载，不阻塞首屏

## 数据分享

- 查询结果仅返回给请求客户端
- 不向第三方出售、共享或转让用户数据
- 不分析或跟踪用户行为用于广告投放

## 合规依据

- 符合 GDPR（通用数据保护条例）要求
- 无需 Cookie 同意弹窗（不使用跟踪 Cookie）
- 用户无需提供任何个人信息即可使用服务

## 安全措施

- 所有 API 请求经 HTTPS 加密传输
- 日志轮转自动清理旧日志（30 天保留期）
- 最小权限原则：进程以专用用户运行，文件系统只读
